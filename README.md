# Market-Stability-Simulator

## Описание
Market Simulator - это инструмент для анализа и симуляции финансовых рынков с использованием агентно-ориентированных моделей. Проект предоставляет возможность исследовать динамику рынка и влияние различных экономических явлений, таких как "эффект горячей картошки".
